# Demo Script

허가온 V2 시연 흐름입니다.

## 1. 앱 진입

로고 랜딩이 짧게 보인 뒤 첫 입력 화면이 나옵니다.

입력 예시:

```text
연남동에서 디저트 카페를 열고 싶어요
```

입력 후 `AI가 분석 중이에요` 로딩 화면이 표시됩니다.

## 2. 정보 수집 루프

질문은 한 번에 하나만 나옵니다.

예상 질문:

- 손님이 매장 안에서 먹고 갈 수 있나요?
- 음식을 직접 만들거나 조리하나요?
- 주류도 판매할 예정인가요?
- 정확한 주소가 정해졌나요?

모르면 `아직 몰라요`를 누르면 됩니다.

## 3. 사전 진단

질문이 끝나면 `준비 방향이 나왔어요` 화면에서 다음을 확인합니다.

- 지금 준비할 서류
- 더 확인할 것
- 부서에 물어볼 것

## 4. 서류 준비

`필요 서류를 준비해요` 화면에서 서류를 체크합니다.

서류 예시:

- 건축물대장 확인
- 건강진단결과서
- 위생교육 수료증

각 서류는 상세를 눌러 이유, 준비 정보, 공식 링크, 발급 순서를 볼 수 있습니다.

## 5. 문의

필요한 문의가 있으면 문의 방법을 고릅니다.

- 전화하기
- 문안 복사
- 방문하기

문의 답변을 입력하면 백엔드가 답변을 분석해 다음 흐름으로 보냅니다.

## 6. 진행 현황

상단 우측 진행 현황 버튼에서 할 일을 확인할 수 있습니다.

- 현재 단계로 돌아가기
- 남은 서류로 이동
- 문의로 이동
- 최근 답변 확인

대시보드에서도 새로 생긴 일과 업데이트된 일을 확인할 수 있습니다.

## 7. 엔딩

모든 서류와 문의가 끝나면 `서류 제출이 끝났어요` 화면으로 넘어갑니다.

중요 조건:

- 서류가 하나라도 남아 있으면 제출 완료 화면으로 가지 않습니다.
- 열린 문의가 있으면 제출 완료 화면으로 가지 않습니다.
- 모든 조건이 끝나야 100% 제출 완료 화면이 나옵니다.
